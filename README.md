# seborgey.github.io

Репозиторий персонального сайта-резюме Сергея Малышев (NLP Research Engineer).

## Описание сайта
- Построен на чистом HTML/CSS с использованием адаптивной верстки.
- Поддерживает динамическое переключение языка интерфейса (русский/английский).
- Включает встроенную функцию экспорта резюме в формат PDF (с использованием библиотеки html2pdf).
- Размещен на хостинге GitHub Pages.
